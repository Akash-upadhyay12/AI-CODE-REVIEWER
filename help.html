🛠️ Help & User Guide

🚀 How to Use

  1. Sign In or continue as a Guest.
  2. Paste your code into the editor.
  3. Select your programming language.
  4. Click on the "Review Code" button.
  5. Wait for the AI analysis.
  6. Check score, bugs, suggestions, and optimized code.

✨ Features

  • AI Code Analysis
  • Code Optimization Suggestions
  • Time Complexity Detection
  • Review History
  • User Rating & Feedback System
  • Edit & Delete Reviews

❓ Frequently Asked Questions

Q1. Which programming languages are supported?

This AI Code Reviewer supports Java Only at that time but later on it also support Python, JavaScript, C++, C and many other popular programming languages.

Q2. Do I need to create an account?

No, you can use the platform as a Guest. However, features like review history and ratings require login.

Q3. Why is the response sometimes slow?

The application uses cloud-hosted AI services, so response times may vary depending on server load and network conditions.

Q4. Can I edit or delete my reviews?

Yes, logged-in users can edit or delete their own ratings and feedback anytime.

📞 Support & Contact

If you face any issues while using the platform, please check the Help section first. You can also share your valuable feedback and suggestions through the Rating section on the About page.

We continuously work on improving the application and adding new features to provide a better user experience.
