About AI Code Reviewer

Project Overview

AI Code Reviewer is a web application that analyzes source code and provides feedback using Artificial Intelligence.

Features

  • AI-powered code review
  • Bug detection
  • Code improvement suggestions
  • Time complexity analysis
  • Review history tracking

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Node.js
  • Express.js
  • MySQL
  • Gemini API

Developer

This project was developed by Akash Upadhyay a B.Tech Computer Science student. The goal of this project is to help developers improve code quality using Artificial Intelligence while gaining hands-on experience with full-stack development, databases, and AI integration.
