About AI Code Reviewer

Project Overview

AI Code Reviewer is a web application that analyzes source code and provides feedback using Artificial Intelligence.

Features

  • AI-powered code review
  • Bug detection
  • Code improvement suggestions
  • Time complexity analysis
  • Review history tracking

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Node.js
  • Express.js
  • MySQL
  • Gemini API

Developer

This project was developed by Akash Upadhyay, a B.Tech Computer Science student, to help developers analyze and improve their code using Artificial Intelligence. It performs AI-powered code reviews, detects bugs, provides optimization suggestions, analyzes time complexity, and stores review history. The application uses Gemini AI as the primary model and automatically switches to Groq AI if Gemini is unavailable, ensuring fast and reliable code reviews. Through this project, I gained hands-on experience in full-stack development, REST APIs, database integration, authentication, and AI model integration.
